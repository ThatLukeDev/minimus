## Run
To run in a VM, the command for qemu is contained within `run`.
Simply run `./run` in a bash shell with dependencies installed to run.

## Build
To build, install all dependencies and run `make`.

## Dependencies
|  Name      |  PKG                |
|------------|---------------------|
|  GNUMake   |  `make`             |
|  GCC       |  `gcc`              |
|  NASM      |  `nasm`             |
|  QEMU      |  `qemu-system-i386` |
